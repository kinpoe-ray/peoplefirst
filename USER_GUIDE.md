# Evolv平台MVP阶段2使用指南

## 概述
Evolv平台MVP阶段2为技能学习平台增加了课程成绩管理和公会聊天功能，让学习更加数据化和社交化。

## 部署地址
🌐 **线上地址**: https://wuv8hlsptqje.space.minimax.io

## 新功能介绍

### 1. 成绩管理系统 📊

#### 功能特性
- **GPA概览**: 显示累计GPA、已完成课程、获得学分、通过率
- **成绩录入**: 支持手动添加各种类型的成绩（测验、作业、项目、考试等）
- **批量导入**: 支持CSV/Excel文件格式，一键导入大量成绩数据
- **数据可视化**: 成绩趋势图表、等级分布、学期对比
- **成绩导出**: 支持将数据导出为CSV格式

#### 使用步骤
1. 点击导航栏中的"**成绩管理**"
2. 在成绩列表中查看所有记录
3. 使用"**添加成绩**"标签页手动录入
4. 使用"**批量导入**"标签页上传文件
5. 在"**数据分析**"标签页查看图表和统计

#### 文件格式要求
CSV格式示例：
```csv
课程名称,分数,满分,学分
高等数学,85,100,4
程序设计,92,100,3
数据结构,78,100,3
```

### 2. 技能公会聊天系统 💬

#### 功能特性
- **实时聊天**: 公会成员可以实时交流
- **聊天室**: 支持多个聊天频道（通用、公告、资源、项目）
- **成员管理**: 查看在线成员、角色权限、活动记录
- **公会动态**: 显示成员加入、消息、成就等动态

#### 使用步骤
1. 进入"**社团**"页面
2. 浏览或搜索感兴趣的公会
3. 点击"**加入公会**"或"**进入公会**"
4. 在聊天弹窗中与其他成员交流
5. 查看"**动态**"和"**成员**"标签页

### 3. 移动端体验 📱

所有新功能都支持移动端访问：
- 响应式布局自动适配屏幕尺寸
- 触摸友好的操作界面
- 优化的移动端导航菜单

## 技术实现

### 前端技术栈
- **React 18**: 现代化组件开发
- **TypeScript**: 类型安全的代码
- **TailwindCSS**: 响应式UI设计
- **Recharts**: 数据可视化图表
- **xlsx**: Excel文件解析
- **papaparse**: CSV文件解析

### 核心组件
- `GradeManagement`: 成绩管理主页面 (939行)
- `GuildChat`: 公会聊天组件 (514行)
- 类型定义: 完整的TypeScript类型系统

### 数据库扩展
- `guild_messages`: 公会消息表
- `guild_activities`: 公会活动表  
- `guild_chat_rooms`: 聊天室表
- `guild_chat_room_members`: 聊天室成员表
